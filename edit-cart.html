Edit Cart

The cart contains all of the parts that a user is considering for purchase. The cart displays the availability of the parts, pricing, quantity, any discounts and applicable taxes. See the figure below for an example.

 

This topic covers:

 

•Using the Cart

•Saving a List

•Add a Comment

•Checkout

 

This figure is an example of a shopping cart in use by a customer. Continue reading below to learn how to use the cart.

 

 

On the cart page, a user can:

 

•Review the items and check their quantities (Qty).

•Change the quantity of each product and click the Update Cart button.

•Remove specific item from the cart.

•Click the Empty Cart button to remove all items.

•Review the Total, which appears below the Update Cart button.

•Save the cart as a public or private Custom List.

•Add message to the dealer in the Comment field.

•Click Go to Checkout to move to the Check out page.

 

Reference column: The little booklet in Refr. column indicates that the item is a part that was placed into the cart from a catalog page. Click the small booklet icon to go back to that catalog page.

 

Continue reading below to learn more about interacting with the cart.

 

Using the Cart

Just above the cart listing, you'll find these two drop-down menus:

 

•Delivery Method – The default is Deliver to me, but you can choose other options (such as Air Freight or Pickup).

•Select Your Warehouse – If you choose in Delivery Method to pickup the part, then here you can choose the warehouse at which you want to pickup the order.

 

 

Review the items in the cart listing and review the values in each of the columns:

 

•Part No – Click the link to go to the product page for that part.

•Stock column – Hover over icon to see the status of the item.

•Quantity – Change quantity by entering a new value in the Qty column.

•Removing an item from the cart – Click the small trash can icon to remove an item from the cart.

•Part details – Click the link in the Part No column to see more details about the item. The Product Details page for that part will appear. To return to the cart, click the Mini cart button (near the search field), and then click Edit Cart.

 

Cart Total

This section appears just below the cart listing. As you make changes to the cart, the value in the Total section will update accordingly.

 

Empty the cart

To permanently remove all items from the cart, click the Empty cart button.

 

 

Saving a List

The user can save the contents of the cart as a Saved List, which can make it easier to duplicate the same order in the future.

 

The figure below shows the Save Order to Custom List section of the cart page. To create a custom list, enter a meaningful name for the new list. Then, select either Public or Private (a private list is hidden from other users). Click the Save button to save the list. To learn how to use this saved list in the future, see Saved List.  

 

 

 

Add a comment

To communicate with the dealership concerning this order, you can enter a note in the Comment section at the bottom of the page. An example of a comment might be "Deliver to the back door."

 

Checkout

To proceed to check out, click the Go to Checkout button.
