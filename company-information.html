Company Information

This page displays company information in several sections, including:

 

•MAIN

•BILLING ADDRESS

•SHIPPING ADDRESS - this figure shows an example of the MAIN section of the Company Information page. This information is drawn from your user profile and also your company profile in the dealer system.

 

Continue reading below to learn how to make changes on this page.

 

 

 

MAIN company information

At the top of this section, you'll see three elements of information:

 

•Company Name – The parent company name.

•E-mail – In most cases, this will be the fleet manager.

•Selling Company Name – The entity that is selling the parts.

 

None of this information above is editable. In the remainder of this section, you'll find many elements of information on which you can make changes.

 

Default Collection Warehouse

If more than one name is available, you can change the name of the warehouse. Click the small pencil icon to display a pop-up from which you can choose another name from the drop-down menu. 

 

Default Delivery Address

If more than one address is available, you can change the address of the physical address. Click the small pencil icon to display a pop-up from which you can choose another address from the drop-down menu.

 

Order Reference Mandatory

You can specify that all orders must include a reference number, such as a PO number. To do so, click the small pencil icon to display a pop-up from which you can choose Yes to enforce the entry of a reference number.

 

Auto Populate My Fleet Limit

This will display VINs that you have marked as favorites that have been set up in Manage My Fleet. If no favorites have been marked–and your fleet has less than 50 Mack or Volvo trucks–then all of your VINs will appear. If your fleet has more than 50 Mack or Volvo trucks, no VINs will appear but you will see the message Click here to Manage My Fleet. To do so, click the small pencil icon to display a pop-up from and enter a new value for the maximum number of units that will appear.

 

Users can override the settings below: Though this sets the default for all PartsASIST customers for this dealership, other users can change this setting in Your Profile.

 

Part Numbers

This setting governs which parts appear in search results and in what order. Click the small pencil icon to display a pop-up from which you can make a selection from a drop-down menu.

 

Choose from one of the following:

•Not selected – (Same as Volvo and Mack) Both Volvo and Mack parts will appear in search results, with Volvo parts appearing first.Only Volvo parts will appear in search results

•Volvo parts – Only Volvo parts will appear in search results.

•Mack parts – Only Mack parts will appear in search results.

•Volvo and Mack – Both Volvo and Mack parts will appear in search results, with Volvo parts appearing first.

•Mack and Volvo –  Both Volvo and Mack parts will appear in search results, with Mack parts appearing first.

 

Measurement Systems

This setting governs which measurement units will display in all PartASIST pages. Click the small pencil icon to display a pop-up from which you can make a selection from a drop-down menu.

 

Choose from one of the following:

•Not selected – (Same as Imperial) Imperial values and units will appear, wherever applicable, in all PartASIST pages.

•Imperial – Imperial values and units will appear, wherever applicable, in all PartASIST pages.

•Metric – Metric values and units will appear, wherever applicable, in all PartASIST pages.

 

Billing Address

This billing address information is drawn directly from the the dealer management system.

 

Shipping Address

This shipping address information correspond to the selection in Default Delivery Address above.
