Company User Management

As a primary user in PartsASIST, you can invite others from your company to join PartsASIST. You can only invite a person who has a current Trucks Customer Portal/Fleet Impact ID that is associated with your company. If they do not, you'll need to request access for any person you invite in the Fleet Impact Maintenance Form that can be found in the Trucks Customer Portal.

 

Inviting a new company user

After verifying that the the person you're inviting has an ID, login in to PartsASIST. Then select Company User Management from the Customer Services menu.

 

 

 

In the New User Invitation page, you'll see the name of your company (Buying Company) and a drop-down list that contains all of the available users in your company (that have an ID).  Choose a User from the drop-down list to see the User E-mail address. If necessary, you can enter an Override User E-mail address.

 

At the bottom of the page, you'll see a list of users from your company who have already been invited to PartsASIST. This list will be empty if no users have been invited.

 

 

 

Send an invitation

To send an invitation, choose a user from the User drop down menu, and look for an email address to appear in User E-mail. If there is no email address, enter the email address of the user in the Override User E-mail field. Click the Send button, and you'll soon receive a confirmation that the invitation has been sent. The user will need to click on the link in the invitation email message and click on Agree to Terms and Conditions. This user can now use PartsASIST as part of your company.

 

 

Invitation status and user type

To see the status of all invitations, review the Status and User Type in the invitation listing at the bottom of the page. The User Type, which is either Normal or Primary, corresponds to the Trucks Customer Portal/Fleet Impact profile of the user.

 

Normal users may use PartsASIST and place orders. A Primary user (see below) can restrict a normal user by (a) setting order limits or (b) deactivating access to PartASIST (by clicking on the small trash can icon).

 

Primary users can also use PartsASIST and place orders. In addition, a primary user has administrative privileges. This includes the ability to invite and delete other users, and also setting order limits for normal users through the Approval Tree process (see the section below). It is not possible to delete a primary user in PartsASIST. To remove a primary user, it is necessary to use the Fleet Impact Maintenance form in the Trucks Customer Portal.
