Company User Management

As a PartsASIST administrator, you can send an invitation to others within your fleet to join PartsASIST from the portal.

 

Missing user? If a user does not appear in the User drop-down in the new user invitation section, this means that the person you want to invite doesn't have a user ID for Trucks Customer Portal/Fleet Impact. You'll need to add that individual by using the Fleet Impact Maintenance Form, which can be found in the Trucks Customer Portal.

 

 

Others in your fleet that have Trucks Customer Portal / Fleet Impact IDs will appear in the User drop-down listing (see the indicator in the figure below). Continue reading below to learn how to send a welcome message to new PartsASIST users.

 

 

 

Send an invitation to join the fleet account in PartsASIST

Follow these steps to add a user:

 

1.Choose a User from the drop-down (see the indicator in the figure below).

2.The User E-mail will display an email address for the user you have chosen. If this is the correct email address, continue to step 4 below. If there is no email address, or the user wants to use another email address in the Override User E-mail field.

3.In Override User E-mail field, enter the email address to send the PartsASIST invitation to an email address other than what appears in User E-mail.

4.Click the SEND button to send the invitation to the new user (see the other indicator in the figure below).

 

The user will receive an email message containing a registration link. When the user clicks on that link, s/he will get a prompt to accept the terms and conditions. After accepting, s/he can complete the registration. A Status of Accepted will appear in the listing at the bottom of the page. The status for any other users will be Pending.

 

Users can invite others: Only the PartsASIST administrator can invite other users in the fleet to join the fleet account in PartsASIST.
