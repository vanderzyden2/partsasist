Manage My Fleet

Fleet information in PartsASIST is arranged by model, model year, and VIN.

 

Information on new trucks or sold trucks will automatically load into PartsASIST according to activity that corresponds to your Customer Portal and Fleet Impact ID. It is managed by the Fleet Administrator using the Fleet Impact Maintenance form in the Trucks Customer Portal. Once new information is provided, it will normally update within 10 business days.

 

On this page, you can:

 

•Designate favorites that will appear on the My Fleet popup.

•View fleet information that has been loaded into your PartsASIST account.

•Add unit numbers to correspond to each of the VIN numbers.

•For efficient editing of a large set of information, download the information on this page into an Excel file.

•Upload an Excel file containing modified information for your fleet.

 

Learn below how to edit information on this page–either by manual edits on the page or by uploading an Excel file.

 

Designate favorites

In a fleet that has numerous trucks spread across different regions, it may be advantageous to mark certain trucks that correspond to a specific location as a favorite. This will allow searching for a particular truck, quick and easy.

To designate favorites that will appear on the My Fleet popup, simply click the star next to the model number or the VIN number. It will change to a darker color to indicate that it is now a favorite. Click the star next to the Model and Year to mark all VINs of that Model and Year as a favorite. Click the star next to one or more VINs to mark those VINs as favorites.

 

 

 

Adding Unit Numbers

In a new account, all of the unit numbers will be empty. You can add a unit number to correspond to each VIN. Simply enter the unit numbers (see the figure below) and then click the SAVE button at the bottom of the page to commit your changes.

 

 

 

Make changes in an Excel file and upload the file

To make changes for every VIN in your fleet, it can be much more efficient to download the information to an Excel file and make the changes there.

 

Follow these steps:

 

1.At the bottom of the page, click the DOWNLOAD EXCEL FILE button.

2.Open the Excel file, in which you can make changes to any of the information, add new trucks, or delete existing trucks.

3.Save the file to your computer.

4.Come back into PartsASIST and click UPLOAD EXCEL FILE.

5.In the popup window, click the Select File button to locate the file on your computer, then click the UPLOAD FILE button.

6.The information on the Manage My Fleet page will reflect the changes that you made in the Excel file.
