Home Page

The home page is the main page for the PartsASIST web store, and is the first page that appears when the clicking the PartsASIST icon on the Trucks Customer Portal.

 

The home page provides all of the primary access to all aspects of PartsASIST. As shown in the figure below, there are several areas in the home page:
 

•Header

•Top navigation – Includes PRODUCT, ORDERS, and CUSTOMER SERVICES

•Content area – Access various features of PartsASIST

•Footer

 

Continue reading below for more details. Other topics in this user guide will refer to these areas of the home page, so it's important to become familiar with all of the details.

 

 

 

Header

At the top of the page, the header contains the following:

•Language drop down – change the language of the site if more than one translation is available.

•Dealer – The name of the dealer from which the user is purchasing parts.

•User name – appears on the right side of the header.

 

 

 

Navigation bar

Below the header and logo, the navigation bar presents the following items. Learn more about these items in Navigation bar.

 

•PRODUCTS

•ORDERS

•CUSTOMER SERVICES

•Search

•Mini cart

 

 

Content Area

The large area in the center of the home page contains information and images. Click on an image to navigate to the content page shown in that image.

 

 

Footer
