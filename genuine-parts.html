The top-level organization for the genuine parts catalog is by function group. Functions groups include Engine, Electric System, Transmission, and others. These can be seen in the menus by clicking Products > Genuine Parts.

 

 

 

Click on Engine to access the page for function group 2. The subcategories for the function group appear in the main panel. Here, we see Engine, Lubricating system, Fuel system, and others.

 

 

Filters

Notice the filters on the left side of the page. These drop-down contain values that correspond to your fleet. You may start from the top and work down, or use any combination of these filters.

 

•Brand – Volvo or Mack

•Model – Model of the truck (VN, VHD, VNR). For Mack, use CXU, GU, AN.

•Model Year – the model years for your fleet.

•Chassis Type – enter or choose a marketing code (identifies the chassis).

•Chassis Number – enter or choose a number (these are the last 6 of the VIN)

•Unit numbers – enter or choose an identifier for a truck in your fleet.

•VIN – If you need to search by VIN, enter or choose a VIN value.

 

After making your choices, click the Apply Filter button.

 

NOTE: By selecting a Unit number or VIN, you'll see the catalog for that specific truck.

 

 

Function group navigation

Below the filters, you'll find a list of all function groups. You'll see that the function group shown in the main panel also appears in the menu. Click on any of these to navigate directly to that function group and reset the filter. You can continue navigating in this menu, or use the links that appear in the main panel.

 

 

Locate a part in the catalog with function groups

Let's illustrate how to find details for a part by navigating through a function group. This section assumes that you have at least one unit number in your Fleet. First, select a Unit Number, as shown in the figure.

 

 

 

In the function group menu (see the section above), click Transmission to display all the subcategories.

 

 

Notice that these subcategories also appear in the main panel.

 

 

 

In either the main panel or the menu, click Clutch, then click Mechanical Clutch, and then click Clutch Complete. At least one part diagram should appear in the main panel. Click the Clear Filter button to start over.

 

 

 

Click the diagram to view the part detail page. Here, you'll find a list of all parts for the component. Learn more in Parts Detail.
