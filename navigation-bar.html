Navigation bar

Below the header and logo, the navigation bar presents the following items:

•Products menu

•Orders menu

•Customer Services menu

•Search field

•Mini cart button

 

 

Detail explanations are given for each of these below.

 

 

Products menu

Click PRODUCTS to display a menu containing:

•☆ MY FLEET – displays a page containing VINs that the user has identified as favorites.

•ALL MAKES – displays all makes products by category.

•GENUINE PARTS – opens genuine parts catalog for registered fleet-specific VINs.

•PROMOTIONS – presents all of the current and upcoming promotions.

•LOCAL CATALOG – access to the dealer-specific catalog (if applicable).

•FLEET CATALOG – access to the fleet catalog (if applicable).

 

 

 

Orders menu

Click ORDERS to display a menu containing:

•EDIT CART – access the shopping cart.

•ORDER INFORMATION – all information about placed orders and order history.

•QUICK ORDER – a fast method for adding parts to the cart, if you know the part number.

•SAVED LIST – a list of parts that were previously saved for re-use.

•UPLOAD FILE – upload a CSV file containing parts that will become part of an order. The CSV file can be prepared in Microsoft Excel or within a fleet management system.

 

 

 

 

Customer Services menu

Click CUSTOMER SERVICES to display a menu containing:

•YOUR PROFILE – information about the customer.

•COMPANY INFORMATION – view and manage company information.

•COMPANY USER MANAGEMENT – manage users (for dealer administrators only).

•APPROVAL TREE – set approval limits for authorized users within the fleet.

•TERMS AND CONDITIONS – review the current PartsASIST terms and conditions.

•MANAGE MY FLEET – set favorites and specify fleet unit numbers to correspond with fleet VIN numbers.

•CONFIGURE UPLOAD FILE – setup the format for the Orders file upload. 

•CONTACT US – send a message to the dealer.

 

 

 

Search

By entering one or more words (or numbers) in this field, the user can search for matches among the part numbers (whole or partial), suppliers, and all part(s) descriptions.

 

 

 

Mini cart

Click this small button to display a pop-up window containing the current contents of the shopping cart. From there, the user can click a link to navigate either to edit the shopping cart or check out page.
