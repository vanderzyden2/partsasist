Check out

The check out is the final step in submitting an order. All prices, discounts and tax are added and the final cost for the order is summarized. The user can check out by either of these methods:

 

•In the Mini cart, click Check out

•On the Edit Cart page, click Go to Check out  

 

This is an example of how the Checkout page appears.

 

 

 

On the Check Out page (see the figure below for an example), the user can:

•Review the Order Summary.

•Optionally, add a Comment to be sent to the dealer with the order.

•Review the Total.

•Enter a Coupon Code, if you a valid code.

•In Your order reference, enter a PO number or other identifier.
 

NOTE: If the setting in Customer Services > Company Information > Order Reference Mandatory is Yes, then it is necessary to add a purchase order number in this field.  

 

•Choose a Payment method.

•Review the Shipping method, which will reflect the choice made back in the Cart. You can change it using the drop-down menu.

•Review the Billing and Shipping information.

•Check the box to agree to terms and conditions.

•Click the Back button to return to the previous page.

•Click the Check out button to finalize the order. 

 

If the user clicks the Check out button, the purchase will be proceed and an order confirmation will appear with a link to the order information.
