Saved List

This feature makes it easy to reorder identical or similar orders by using order templates. An example of such an order would be the set of all filters for a specific truck or a type of truck.

 

A saved list is created in the Save Order to Custom List feature in the cart. Learn more in the Saving a List section of Edit Cart.

 

There are three types of saved lists:

 

•Private – saved lists for the user that created the list.

•Public – these are saved lists that users within the organization can see.

•Dealer – these are saved lists that dealers have made visible to their PartsASIST users.

•Volvo – these are lists created by Volvo that are made available to all PartsASIST users.

 

Below is an example view of the Saved List page. On this page, you can:

 

•Click the name of the saved list to view details (and add it to the cart from there).

•Click the small trash can icon to delete a saved list.

•Click the Add to Quick Order button to begin a Quick Order, and proceed from there to the Cart.
