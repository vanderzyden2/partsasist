Accessing PartsASIST

Trucks Customer Portal

The PartsASIST web store is accessible through a link on the Volvo Trucks Customer Portal website. See the large PartsASIST icon in the figure below.

 

 

 

User ID and Password

PartsASIST uses a single sign-on (SSO) feature. The customer ID and password to access PartsASIST are the same as those for accessing the Trucks Customer Portal. Click the large PartsASIST icon to open the main page. If your user profile includes access to Fleet Impact, the registered VINs for Fleets will automatically populate in the PartsASIST web store. Learn more in Manage My Fleet.

 

No ID and password? If you do not have a Trucks Customer Portal ID and password, please contact your local Volvo dealer to request one.

 

 

Dealer Invitation

So that you can see pricing, dealer inventory, and place order, it is necessary for your local dealer to invite a customer to join their PartsASIST web store. An email invitation with an activation link will be sent to the primary email address in the customer profile. When user account activation is complete, the customer will access PartsASIST through the Trucks Customer Portal.
