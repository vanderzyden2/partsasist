Cart

The cart contains all of the parts that a user is considering for purchase. In the cart, the availability of the parts is shown, along with any discounts and applicable taxes. See the figure below for an example.

Cart Overview

On the cart page, as user can:

 

•Review the items and check their quantities (Qty)

•Change the quantity of each product and click the Update Cart button

•Remove specific item from the cart

•Click the Empty Cart button to remove all items

•Review the Total, which appears below the Update Cart button

•Save the cart as a public or private Custom List

•Add message to the dealer in the Comment field

•Click Go to Checkout to move to the Check out page

 

Reference column: The little booklet in Refr. column indicates that the item is a part that was placed into the Cart from a catalog page. Click the small booklet icon to go back to that catalog page. If a user adds a part to the cart from the search page, no booklet icon will appear for that item in the cart.

 

Continue reading below to learn more about interacting with the cart.

 

 

Using the Cart

Part details: Click the link in the Part No column to see more details about the item. The Product Details page will appear.

 

Update the quantity: To update quantity of an item, enter a new value in the Qty field for that item and click the Update Cart button at the bottom of the listing.

 

Remove a single item: Click the small basket icon for that item.

 

Empty the cart: To permanently remove all items from the cart, click the Empty cart button.

 

To see if a specific product has some quantity discounts it’s necessary to place the mouse pointer over the corresponding value in Sum column – a small pop up will be displayed with the discount(s) information.

 

Save the Cart to a Custom List

The user can save the contents of the cart as a custom list that can make it easier to duplicate the same order in the future.

 

The figure below shows the Save Order to Custom List section of the cart page. To create a custom list, enter a meaningful name for the new list. Then, select either Public or Private (the list is hidden from other users). Click the Save button to save the list. Later on the saved list can be used to place an order.
