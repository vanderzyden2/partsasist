Advanced Search & Filtering

As seen in the examples of the Quick Start articles in this guide, customers can use the fleet filter feature to search for parts that are for a particular truck. Those examples show how easy it is to apply a filter before starting your search.

 

Quick Start articles on search – If you haven't already, be sure to review these tutorial articles about different ways to do basic searches:

 

•Quick Start - Search in the Search bar

•Quick Start - Search from the Product Menu

•Quick Start - Search from Catalog Icons

 

But let’s say you began a search in the Genuine Parts catalog and forgot to apply a filter for a particular Unit number. You've filtered all the way down through the function groups and find yourself on a detailed parts page that contains multiple part number options for an instrument cluster–similar to what is shown in the figure below. How do you determine which one to order for a particular truck? Continue reading below to learn how.

 

 

 

In the Fleet filter section on the left hand side of the screen, click the Unit drop-down arrow (or VIN drop-down arrow if you have not added Unit numbers) box.

 

 

When you click the Unit drop-down, you'll see a list of all the unit numbers that can accept the part numbers shown in the parts detail.

 

 

 

Choose one of the unit numbers and click the Apply Filter button. The parts detail will now display only one instrument cluster part number, which is the one that fits the truck corresponding to the unit number that you chose.

 

From here, you can check the box next to the part number and click the Add to Cart button. Next you may want to edit the cart.
